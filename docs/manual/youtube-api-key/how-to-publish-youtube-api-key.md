# YouTube Data API 키 발급 가이드

Mark Explorer에서 유튜브 재생목록 정보를 자동으로 가져오거나 상세 정보를 표시하기 위해서는 Google의 **YouTube Data API v3** 키가 필요합니다. 아래 절차에 따라 안전하게 API 키를 발급받고 설정할 수 있습니다.

---

### 1. Google Cloud 프로젝트 생성
1. **[Google Cloud Console](https://console.cloud.google.com/)**에 접속하여 Google 계정으로 로그인합니다.
2. 화면 상단의 **프로젝트 선택** 드롭다운을 클릭한 후, 팝업창 오른쪽 상단의 **[새 프로젝트(New Project)]**를 클릭합니다.
3. 프로젝트 이름(예: `Mark-Explorer-Youtube`)을 입력하고 **[만들기(Create)]** 버튼을 누릅니다.

### 2. YouTube Data API v3 활성화
1. 생성된 프로젝트가 선택된 상태인지 확인합니다.
2. 왼쪽 탐색 메뉴에서 **[API 및 서비스] > [라이브러리]**를 선택합니다.
3. 검색창에 `YouTube Data API v3`를 입력하고 검색 결과에서 해당 항목을 클릭합니다.
4. **[사용(Enable)]** 버튼을 클릭하여 API를 활성화합니다.

### 3. API 키(Credentials) 발급
1. API 활성화 후 나타나는 대시보드 또는 왼쪽 메뉴에서 **[사용자 인증 정보(Credentials)]** 탭으로 이동합니다.
2. 상단의 **[+ 사용자 인증 정보 만들기(Create Credentials)]** 버튼을 누르고 **[API 키(API key)]**를 선택합니다.
3. 팝업창에 생성된 **API 키**를 복사하여 안전한 곳에 보관하세요. (이 키를 앱 설정에 입력하게 됩니다.)

### 4. 보안 설정 (필수 권장)
생성된 API 키가 타인에게 도용되는 것을 방지하기 위해 사용 범위를 제한해야 합니다.

1. 생성된 API 키 우측의 **수정(연필 모양 아이콘)**을 클릭합니다.
2. **애플리케이션 제한사항 (Application restrictions)**:
   - **[없음(None)]**을 선택합니다. 
   - *이유: Mark Explorer는 로컬 환경 및 다양한 네트워크에서 실행되므로, 특정 도메인이나 IP로 제한하면 앱 작동이 멈출 수 있습니다.*
3. **API 제한사항 (API restrictions)**:
   - **[키 제한(Restrict key)]**을 선택합니다.
   - 드롭다운 메뉴에서 **`YouTube Data API v3`**를 찾아 체크한 후 [확인]을 누릅니다.
   - **※ 주의**: *'서비스 계정을 통해 API 호출 인증'* 이라는 체크박스가 보일 수 있는데, 이는 **체크하지 마세요.** (유튜브 API에는 필요하지 않은 설정입니다.)
   - *이 설정을 통해 이 키가 유튜브 API 호출 외의 용도로 도용되는 것을 차단합니다.*
4. 하단의 **[저장(Save)]** 버튼을 클릭하여 설정을 완료합니다.

---

> [!IMPORTANT]
> **주의사항**
> - **보안**: API 키는 비밀번호와 같습니다. 공개된 저장소(GitHub 등)에 키가 노출되지 않도록 주의하세요.
> - **할당량**: YouTube Data API v3의 기본 무료 할당량은 일일 10,000 유닛입니다. 일반적인 개인 용도로는 충분하지만, 대량의 검색이나 재생목록 호출 시 할당량이 소진될 수 있습니다.
