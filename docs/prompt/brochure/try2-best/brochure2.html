Mark Explorer는 로컬 마크다운 파일을 지능적으로 탐색하고 실시간으로 편집하는 프리미엄 도구입니다. Typora의 직관성, VSCode의 효율성, Obsidian의 깊이를 하나로 담았습니다.

강력한 하이브리드 워크플로우

단순한 에디터를 넘어, 당신의 지식 베이스를 위한 최고의 파트너

Hybrid Editor

Tiptap 기반의 실시간 라이브 에디터와 2x2 분할 화면, 멀티탭 시스템을 통해 복잡한 문서 작업도 쾌적하게 수행합니다.

Smart Link Cards

일반 URL을 노션 스타일의 썸네일 카드로, 유튜브 링크를 인앱 재생이 가능한 비디오 모드로 자동 변환합니다.

AI & MCP Integration

Gemini AI가 문서의 맥락을 이해하고, MCP 프로토콜을 통해 외부 도구와 데이터를 연동하여 지능적인 제안을 제공합니다.

Nextra Export

클릭 한 번으로 로컬 디렉터리를 Nextra 기반의 정적 사이트로 익스포트하여 당신의 지식을 세상과 공유하세요.

Perfectly Responsive

PC는 물론 안드로이드, iOS 모바일 환경까지 완벽하게 지원하며 Safe Area와 모바일 전용 드로어를 제공합니다.

Plugin System

사용자 정의 템플릿과 플러그인 아키텍처를 통해 당신만의 독특한 마크다운 환경을 구축할 수 있습니다 (준비 중).

당신을 위한 사용 사례

Mark Explorer는 다양한 라이프스타일에 맞춰 유연하게 변화합니다.

Mobile Knowledge Manager

출퇴근 길이나 이동 중에도 스마트폰에서 마크다운 파일을 손쉽게 관리하세요. 구글 드라이브나 iCloud와 연동하여 언제 어디서나 끊김 없는 편집이 가능합니다.

  • 모바일 전용 UI/UX 최적화
  • 클라우드 디렉터리 완벽 연동

YouTube Content Curator

학습용 유튜브 영상의 핵심 내용을 기록하고, 영상 썸네일을 마크다운 내에 배치하여 시각적인 요약 노트를 만드세요. 나만의 유튜브 지식 저장소가 탄생합니다.

  • 유튜브 링크 썸네일 자동 추출
  • 인앱 비디오 플레이어 지원

Privacy-First Journaling

등산 기록이나 일기 같은 소중한 일상을 소셜 플랫폼 대신 안전한 로컬 파일로 보관하세요. 백업 걱정 없이 맑은 정신과 그 당시의 바이브를 기록할 수 있습니다.

  • 100% 로컬 데이터 제어
  • 오프라인 우선(Offline-First) 설계
