# Explorer 컨텍스트 메뉴 위치 수정 목표 (GOAL.md)

탐색기(Explorer) 내 폴더 및 파일 우클릭 시 발생하는 컨텍스트 메뉴 위치 이탈 버그를 해결하고 사용자 경험을 개선합니다.

## 🎯 최종 목표
- [x] 탐색기 항목(파일/폴더) 우클릭 시, 마우스 커서 바로 옆에 메뉴가 나타나는가?
- [x] 윈도우 스크롤이나 사이드바 크기에 상관없이 정확한 위치에 고정(Fixed)되는가?
- [x] 다른 파일을 우클릭했을 때 기존 메뉴가 닫히고 새 위치에 메뉴가 나타나는가?

## ✅ 성공 기준
1. **좌표 일관성**: 모든 탐색기 항목에서 우클릭 위치와 메뉴 시작점이 일치해야 함.
2. **레이아웃 독립성**: 부모 컨테이너(`relative`)의 영향을 받지 않는 `fixed` 포지션 사용.
3. **시각적 완성도**: 메뉴가 나타날 때 깜빡임이나 위치 튐 현상이 없어야 함.
