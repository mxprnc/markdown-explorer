### 2단계: URL 변환 로직 고도화 (Editor.web.tsx)
- **변환 시점 조정**: 
    - `LiveMarkdownExtension` 내의 `isTouching` 로직에서 `$from.pos`와 `$to.pos`가 URL의 *중간*에 있을 때만 텍스트 상태를 유지하도록 변경.
    - 즉, 커서가 URL의 시작 바로 앞이나 끝 바로 뒤에 있을 때는 노드로 변환되도록 하여 사용자가 입력을 마치자마자 프리미엄 기능을 볼 수 있게 함.

### 3단계: 통합 테스트

- 긴 URL 입력 시 팝업 레이아웃 확인.
- 커서 이동에 따른 텍스트 <-> 노드 전환의 부드러움 확인.
