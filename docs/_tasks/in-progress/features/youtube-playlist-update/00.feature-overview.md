# [Feature Overview] YouTube Playlist Exporter 고도화

**Feature Name**: `youtube-playlist-update`
**Version**: 1.0 (GA)
**Completion Date**: 2026-05-04
**Status**: ✅ **Implementation & Verification Completed**

---

## 1. 소개 (Introduction)
기존의 YouTube Playlist Export 기능을 대폭 개선하여, 사용자에게 **정직한 피드백(Honest Preview)**과 **정밀한 제어권(Manual Control)**을 제공합니다. 특히 대규모 플레이리스트 관리 시 발생하던 불투명한 추출 과정을 제거하고, 실제 결과물과 동일한 환경에서의 미리보기를 지원하여 신뢰성을 확보했습니다.

## 2. 핵심 기능 (Key Features)

### 🚀 Expanded Extraction Modal (대형 다이얼로그)
- **스케일업 UI**: 화면의 **90% x 85%**를 활용하는 대형 모달을 통해 쾌적한 설정 및 미리보기 환경을 제공합니다.
- **실시간 라이브 프리뷰**: URL 입력 시 YouTube API를 통해 실제 데이터를 불러와, **Markdown** 원문과 **Preview(렌더링)** 결과물을 탭을 통해 실시간으로 확인할 수 있습니다.

### 🛠️ 정밀한 추출 제어 (Precision Control)
- **수동 ITEM LIMIT 입력**: 고정된 개수 선택 방식에서 벗어나, 사용자가 원하는 추출 개수를 직접 입력할 수 있는 `TextInput` 방식을 채택했습니다. (기본값: 20)
- **최적화된 기본값**: 가장 선호되는 **List Type: Plain**을 기본값으로 설정하여, 기호 없는 정갈한 텍스트 나열을 즉시 제공합니다.
- **2-Tab 체제 단순화**: 복잡한 편집 기능을 제거하고 `Markdown | Preview`의 2단 구조로 집중도를 높였습니다.

### 📂 탐색기 연동 (Context Menu Integration)
- 탐색기 폴더 우클릭 시 `Create Youtube Playlist Note` 메뉴를 통해 즉시 전용 노트 생성이 가능하며, 해당 폴더가 타겟 디렉토리로 자동 지정되는 빠른 워크플로우를 지원합니다.

## 3. 기술적 특징 (Technical Highlights)

### 🛡️ Playwright 기반 E2E 자동 검증
- **검증 범위**: Chromium, Firefox, Webkit 3개 브라우저에서 초기값 설정, 수동 입력 동작, 탭 전환, 프리뷰 렌더링을 자동화 테스트로 완벽 검증했습니다.
- **결과**: **15개 테스트 케이스 100% 합격**.

### 🔗 실재 데이터 바인딩 (Real Data Binding)
- Mock 데이터를 제거하고 실제 YouTube Data API v3를 연동하여 플레이리스트의 메타데이터(제목)와 아이템 정보를 동적으로 수집합니다.
