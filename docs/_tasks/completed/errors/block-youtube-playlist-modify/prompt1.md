## Idea 1

블록으로 선택된 영역에 Youtube 영상 또는 웹사이트 링크가 있을때 해당 블록을 우클릭하면 컨텍스트 메뉴가 나타나고 'Link Type' 메뉴에서 유튜브 링크는 'Thumb'. 'Link', 'Video', 'Plain'을 로 바꿔서 선택 할수 있도록, 일반 웹사이트 링크는 'Video' 를 제외한 다른 타입으로 변경할수 있도록 하는 기능 수정에 대한 프롬프트를 바로 아래의 '### 프롬프트' 에 작성하세요. 위의 내용은 수정하거나 삭제하지 마세요.


### 프롬프트

에디터(Editor) 모드에서 텍스트 블록을 선택하고 우클릭했을 때, 선택 영역 내에 포함된 유튜브 링크나 일반 웹사이트 링크의 표시 형식을 한꺼번에 변경할 수 있는 'Link Type' 기능을 구현하세요.

**1. 기능 요구 사항**
- **링크 감지**: 선택된 텍스트 영역 내에서 `[mx-...]` 패턴의 커스텀 링크 노드나 일반적인 `[텍스트](URL)` 마크다운 링크를 모두 감지해야 합니다.
- **컨텍스트 메뉴 확장**: `EditorContextMenu.tsx`에 'Link Type' 섹션을 추가하세요. 선택 영역에 링크가 하나 이상 포함된 경우에만 이 메뉴가 노출되어야 합니다.
- **타입 전환 옵션**:
  - **유튜브 링크 (YouTube URL 감지 시)**: 'Card(Thumb)', 'Link', 'Video', 'Plain'의 4가지 옵션을 제공합니다.
  - **일반 웹 링크**: 'Card(Thumb)', 'Link', 'Plain'의 3가지 옵션을 제공하며, 'Video' 옵션은 비활성화하거나 제외합니다.
- **일괄 변환 로직**: 사용자가 특정 타입을 선택하면, 선택 영역 내의 모든 대상 링크를 해당 형식의 마크다운(`[mx-video#제목](URL)` 등)으로 일괄 교체하고 에디터에 반영합니다.

**2. 기술적 가이드**
- **에디터 연동**: Tiptap 에디터의 `state.selection`을 사용하여 선택된 범위의 텍스트와 노드를 가져옵니다.
- **정규식 활용**: `MX_LINK_CARD_REGEX` 및 일반 URL 감지 정규식을 활용하여 링크를 추출하고, 선택된 타입에 맞는 마크다운 문자열을 생성하세요.
- **트랜잭션 처리**: 에디터의 `insertContent` 또는 `tr.replaceWith`를 사용하여 선택 영역의 내용을 새로운 마크다운으로 깔끔하게 교체하세요. 이때 기존의 일반 텍스트는 보존되고 링크 패턴만 변환되어야 합니다.
- **디자인 가이드**: 기존 `EditorContextMenu`의 프리미엄 디자인(ThemeContext 연동, Blur 효과 등)을 유지하며, 서브메뉴나 구분선을 사용하여 'Link Type' 섹션을 명확히 구분하세요.

**3. 예외 처리**
- 선택 영역에 링크가 없는 경우 메뉴를 표시하지 않습니다.
- 변환 과정에서 제목(Alt Text)이 없는 경우 URL을 제목으로 기본 사용하거나 기존 제목을 최대한 보존합니다.
