# 비영어권 파일명 및 경로 처리 최적화 목표 (GOAL1.md)

모바일 업로드 및 크로스 플랫폼 환경에서 한글 등 비영어권 언어로 된 이미지 파일명이 깨지거나 경로 오류가 발생하는 문제를 해결하여 데이터 무결성을 보장합니다.

---

## 🎯 핵심 목표
1.  **파일명 안정성 확보**: 특수문자, 공백, 유니코드(한글 등)가 포함된 파일명이 시스템 간 이동 시 깨지지 않도록 정규화.
2.  **경로 가독성 유지**: 기계적인 해시값보다는 사용자가 식별 가능한 이름을 최대한 유지하는 방향으로 처리.
3.  **플랫폼 간 호환성**: Windows, Mac, Android, iOS 및 정적 사이트 빌드(Nextra 등) 환경에서 링크가 끊기지 않도록 보장.

## ✅ 주요 체크리스트
- [ ] 파일명 내 공백 및 특수문자의 Kebab-case 치환 로직.
- [ ] Mac(NFD)과 Windows(NFC) 간의 한글 자모 분리 현상 방지를 위한 정규화.
- [ ] 비영어권 문자가 포함된 파일명의 URL 인코딩 처리 검토.
- [ ] 이미지 업로드 시 중복 방지를 위한 네이밍 컨벤션 수립.
