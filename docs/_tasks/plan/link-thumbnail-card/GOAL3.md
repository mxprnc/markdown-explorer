# 링크 편집 및 툴바 고도화 목표 (GOAL3.md)

`GOAL2.md`에서 구현한 기초 UI를 바탕으로, Notion과 동일한 사용자 경험을 제공하기 위해 호버 및 편집 로직을 더욱 정교하게 다듬는 것을 목표로 합니다.

---

## 🎯 핵심 목표

1.  **호버 기반 링크 툴바 (Hover Toolbar)**:
    *   링크 노드 마우스 오버 시, 별도의 클릭 없이도 URL 정보와 [복사], [편집] 버튼이 포함된 툴바가 즉시 노출되도록 개선.
    *   사용자가 링크의 목적지(URL)를 바로 확인하고 편집 모드로 진입할 수 있는 직관적인 경로 제공.

2.  **디자인 정교화 (Notion-Like)**:
    *   `link-edit-1.png` 이미지에서 보여지는 Notion 특유의 간결하고 세련된 툴바 스타일 적용.
    *   URL 텍스트에 `JetBrains Mono` 폰트 적용 및 생략(...) 처리 최적화.
    *   아이콘과 텍스트 간의 여백, 버튼의 호버 효과 등을 세밀하게 조정.

3.  **상호작용 안정성 (Interaction Stability)**:
    *   링크와 툴바 사이의 이동 과정에서 메뉴가 깜빡이거나 사라지지 않도록 `Hover Bridge` 로직 강화.
    *   편집 버튼 클릭 시 나타나는 팝업이 툴바와 자연스럽게 연결되도록 배치 및 애니메이션 조정.

## ✅ 주요 체크리스트

- [x] **호버 시 링크 툴바 즉시 노출** (복사, 편집 버튼 포함)
- [x] **툴바 디자인 고도화**: Notion 스타일의 폰트, 여백, 아이콘 적용
- [x] **호버 브릿지 로직 최적화**: 툴바 이동 시 이탈 방지
- [x] **편집 팝업 진입 경로 명확화**: 사용자가 '편집' 위치를 쉽게 찾을 수 있도록 개선
- [x] **상태 관리 리팩토링**: `isHovered`, `showMenu`, `isEditing` 간의 논리적 연결성 강화
