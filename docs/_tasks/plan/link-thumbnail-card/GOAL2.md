# 링크 썸네일/카드 보기 기능 목표 (GOAL2.md)

`GOAL1.md`에서 정의한 기본 기능을 바탕으로, Notion 스타일의 프리미엄 UI/UX를 적용하여 사용자 경험을 고도화하는 것을 목표로 합니다.

---

## 🎯 핵심 목표

1.  **프리미엄 시각 디자인 (Rich Aesthetics)**:
    *   **Glassmorphism**: 백드롭 블러(`backdrop-filter: blur(12px)`)와 반투명 배경을 적용하여 현대적이고 세련된 느낌 제공.
    *   **고급 그림자**: 깊이감 있는 그림자를 통해 UI 요소 간의 계층 구조 명확화.
    *   **세련된 타이포그래피**: `Inter` 폰트와 `JetBrains Mono`를 조합하여 정보의 가독성 및 개발자 친화적 감성 강화.

2.  **정교한 상호작용 (Interactive Excellence)**:
    *   **마우스 오버 브릿지**: 링크와 툴바 사이의 간격에서 마우스가 이동할 때 메뉴가 사라지지 않도록 브릿지 로직 구현.
    *   **부드러운 애니메이션**: Fade-in 및 Scale-up 효과를 통해 UI의 등장이 자연스럽게 느껴지도록 개선.
    *   **키보드 접근성**: `Esc` 키를 통한 팝업 닫기 등 직관적인 키보드 지원.

3.  **향상된 편집 경험**:
    *   **Notion 스타일 팝업**: URL과 별칭을 분리하여 편집할 수 있는 직관적인 팝업 레이아웃.
    *   **상태 동기화**: 편집 시 실시간으로 에디터 노드의 속성을 업데이트하고 마크다운 문법과 동기화.

## ✅ 주요 체크리스트

- [x] **Glassmorphism UI 적용** (툴바 및 편집 팝업)
- [x] **호버 툴바 구현**: 아이콘, URL(생략형), 복사/편집 버튼 포함
- [x] **편집 팝업 구현**: URL 입력, 제목 입력, 삭제/취소/저장 버튼 포함
- [x] **호버 브릿지 로직**: 메뉴 이탈 방지
- [x] **애니메이션 추가**: 진입/퇴장 효과
- [x] **다크 모드 최적화**: 테마별 색상 및 블러 농도 조정
- [x] **반응형 대응**: 모바일 환경에서의 팝업 너비 최적화
