# Video 타입 깜빡임 및 안정성 개선 목표 (GOAL7.md)

비디오 플레이어 로딩 시 발생하는 깜빡임 증상을 해결하고, 호버 툴바의 안정성을 높여 사용자 경험을 프리미엄 수준으로 끌어올리는 것을 목표로 합니다.

---

## 🎯 핵심 목표

1.  **랜더링 성능 최적화 (Rendering Optimization)**:
    *   `iframe` 요소가 불필요하게 다시 그려지지 않도록 메모이제이션 및 안정적인 키 관리를 적용하여 깜빡임 제거.

2.  **상호작용 안정성 (Interaction Stability)**:
    *   비디오 플레이어 위로 마우스가 올라갔을 때 툴바가 사라지거나 깜빡이는 현상 해결.
    *   호버 브릿지 영역을 확장하고 상태 전환 로직을 견고하게 수정.

3.  **레이아웃 일관성 (Layout Consistency)**:
    *   편집 후 상태 전환 시 발생하는 레이아웃의 급격한 변화나 시각적 튀는 현상 방지.

## ✅ 주요 체크리스트

- [x] **비디오 플레이어 컴포넌트 메모이제이션 적용**
- [x] **Iframe 재생성 방지를 위한 안정적인 Key 설정**
- [x] **호버 유지 로직 개선 (Delay 및 Bridge 보강)**
- [x] **편집 완료 후 상태 전환 시각적 노이즈 제거**
- [x] **다크 모드에서의 비디오 영역 스타일 최적화**
