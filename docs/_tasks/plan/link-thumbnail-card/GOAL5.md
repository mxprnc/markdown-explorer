# 링크 복사 알림(토스트) 구현 목표 (GOAL5.md)

링크 복사 기능 사용 시 사용자에게 명확한 피드백을 제공하기 위해 세련된 토스트 메시지 기능을 추가하는 것을 목표로 합니다.

---

## 🎯 핵심 목표

1.  **시각적 피드백 제공 (Visual Feedback)**:
    *   복사 버튼 클릭 성공 시, 화면에 "Link copied to clipboard"와 같은 안내 메시지를 표시하여 동작 완료를 인지시킴.

2.  **프리미엄 디자인 적용 (Premium Design)**:
    *   프로젝트의 'Rich Aesthetics' 가이드라인에 맞춰 유리 질감(Glassmorphism), 백드롭 블러, 부드러운 애니메이션이 적용된 토스트 UI 구현.
    *   단순한 알림을 넘어 앱의 전반적인 감도에 어울리는 세련된 디자인 유지.

3.  **언어 일관성 (Language Consistency)**:
    *   메시지 문구를 영어("Link copied to clipboard")로 작성하여 프로젝트의 기본적인 언어 톤앤매너 준수.
