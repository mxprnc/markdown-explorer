# 미리보기 모드 비디오 안정화 목표 (GOAL12.md)

익스플로러(미리보기) 모드에서 문서를 스크롤하거나 자동 저장이 발생할 때 유튜브 영상이 깜빡거리는 현상을 해결하는 것을 목표로 합니다.

---

## 🎯 핵심 목표

1.  **미리보기 전용 VideoPlayer 컴포넌트화**:
    *   익스플로러 모드의 `a` 태그 렌더러 안에 직접 작성된 `iframe` 로직을 별도 컴포넌트로 분리하고 `React.memo`를 적용하여 렌더링 안정성 확보.

2.  **깜빡임 시각적 차단 (Thumbnail Fallback)**:
    *   에디터 모드와 동일하게 `iframe` 로딩 중에 썸네일 이미지를 배경으로 보여주어 블랙 스크린(깜빡임) 노출 차단.

3.  **컴포넌트 참조 안정성 강화**:
    *   `MarkdownPreview`가 리렌더링되더라도 영상 노드(`iframe`)의 DOM이 파괴되지 않도록 `key` 값과 컴포넌트 구조를 최적화.

- [ ] **a 태그 렌더러 교체 및 동작 확인**
