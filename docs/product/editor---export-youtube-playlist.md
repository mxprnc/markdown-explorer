# YouTube 재생목록 추출 및 렌더링 (Export YouTube Playlist)

이 문서는 에디터(Editor) 내에서 유튜브 재생목록을 자동으로 추출하고, 이를 다양한 형식의 마크다운 카드로 변환하여 렌더링하는 기능에 대해 설명합니다.

---

## 1. 주요 기능 개요

유튜브 재생목록 URL을 입력받아 목록 내의 모든 비디오 제목과 링크를 가져오고, 사용자가 선택한 리스트 스타일(번호 매기기, 글머리 기호, 일반 텍스트)에 맞춰 에디터에 즉시 삽입합니다. 삽입된 링크는 커스텀 노드 시스템을 통해 비디오 플레이어 또는 프리뷰 카드로 자동 변환됩니다.

### 핵심 워크플로우
1. 에디터 우클릭 메뉴 또는 단축키를 통해 **Youtube Extractor** 실행
2. 유튜브 재생목록 URL 입력 및 **List Type** 선택 (Numbered, Bulleted, Plain)
3. 추출된 데이터를 바탕으로 `[mx-video#제목](URL)` 형식의 마크다운 생성
4. 에디터에 삽입 시 Tiptap `linkCard` 노드로 즉시 변환 및 비디오 플레이어 렌더링

## 2. 세부 기능 및 인터페이스

### 2.1 유튜브 추출 모달 (YoutubeExtractorModal)
- **URL 입력**: 유튜브 재생목록(Playlist) 링크를 감지합니다.
- **List Type 선택**:
  - `Numbered`: `1. [제목](URL)` 형식
  - `Bulleted`: `- [제목](URL)` 형식
  - `Plain`: 제목과 링크만 개행으로 구분된 형식
- **자동 마크업**: 추출 직후 에디터의 `[mx-...]` 패턴으로 자동 변환되어 삽입됩니다.

### 2.2 실시간 노드 변환 (Live Rendering)
- **즉시 렌더링**: 추출된 텍스트가 삽입될 때, `editor.commands.enter()` 명령을 강제로 실행하여 커서 이탈 및 노드 변환(Tiptap node creation)을 즉각적으로 트리거합니다.
- **Video 플레이어**: `mx-video` 타입인 경우, 에디터 내에서 유튜브 임베드 플레이어가 직접 노출되어 즉시 재생이 가능합니다.

## 3. 기술적 개선 및 안정화

### 3.1 정규식 동기화 (Regex Synchronization)
- 에디터(`Editor.web.tsx`), 미리보기(`MarkdownPreview.web.tsx`), 유틸리티(`MarkdownUtils.ts`) 간의 링크 카드 감지 정규식을 일치시켜 렌더링 일관성을 확보했습니다.
- 공백 및 특수 기호(#)에 유연하게 대응하도록 정규식을 개선했습니다.

### 3.2 HTML 속성 이스케이프 (Attribute Escaping)
- 유튜브 제목에 쌍따옴표(`"`)가 포함된 경우 HTML `<span>` 태그의 속성이 깨지는 문제를 해결하기 위해, `preprocessMarkdown` 과정에서 제목 데이터를 안전하게 이스케이프(`&quot;`) 처리합니다.

### 3.3 유튜브 ID 감지 개선
- `utils/LinkCardUtils.ts`의 `getYoutubeId` 함수를 고도화하여 Shorts, 모바일용 URL(youtu.be), 임베드 URL 등 다양한 유튜브 링크 형식을 완벽하게 지원합니다.

## 4. UI/UX 디자인
- **프리미엄 컨텍스트 메뉴**: `EditorContextMenu.tsx`를 통해 유튜브 추출 기능을 직관적으로 실행할 수 있습니다.
- **반응형 모달**: 다크 모드와 라이트 모드를 완벽하게 지원하며, 애니메이션 효과를 통해 매끄러운 사용자 경험을 제공합니다.
