# 템플릿 시스템 제품 명세 (template-system.md)

반복적인 마크다운 구조를 효율적으로 관리하고 생산성을 높이기 위한 템플릿 시스템입니다.

## 🌟 핵심 가치
- **반복 작업 최소화**: 회의록, 저널, 코드 스니펫 등을 단 한 번의 클릭으로 완성합니다.
- **포맷의 표준화**: 프로젝트 내에서 일관된 문서 형식을 유지하도록 돕습니다.
- **로컬 중심 관리**: 사용자의 템플릿은 일반 마크다운 파일로 저장되어 관리가 직관적입니다.

## 🛠 주요 기능
1. **로컬 파일 기반 템플릿**
   - `.mark-explorer/templates` 폴더 내의 `.md` 파일들이 자동으로 템플릿으로 등록됩니다.
   - 폴더 구조를 그대로 유지하며 탐색기에서 직접 템플릿을 수정할 수 있습니다.

2. **동적 변수 치환 (Variable Substitution)**
   - 템플릿 삽입 시 특정 키워드를 실시간 데이터로 자동 변환합니다.
   - 지원 변수: `{{date}}` (현재 날짜), `{{time}}` (현재 시간), `{{title}}` (현재 파일명).

3. **삽입 및 관리 도구**
   - **명령어 팔레트**: '템플릿 삽입' 명령어를 통해 빠른 검색 및 삽입이 가능합니다.
   - **전용 사이드바 (개발 예정)**: 시각적인 템플릿 목록과 관리 버튼을 제공합니다.

## 🖥 사용자 워크플로우
1. 사용자가 `.mark-explorer/templates` 폴더에 원하는 형식의 마크다운 파일을 만듭니다.
2. 편집 중인 문서에서 `Cmd/Ctrl + P`를 눌러 '템플릿 삽입'을 선택합니다.
3. 원하는 템플릿을 선택하면 커서 위치에 변수가 치환된 내용이 즉시 삽입됩니다.

## 💡 팁
- 템플릿 상단에 YAML Frontmatter를 추가하여 메타데이터를 함께 관리할 수 있습니다.
