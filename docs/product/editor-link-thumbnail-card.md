# Editor Link Card 시스템 사양서 (Product Specification)

이 문서는 Mark Explorer 에디터에서 지원하는 **Link Card (Link Type)** 시스템의 기능과 사양을 정의합니다.

---

## 1. 개요
사용자가 입력한 일반 URL을 시각적으로 풍부한 카드 UI로 변환하여 보여주는 기능입니다. Notion 스타일의 카드 인터페이스를 지향하며, 에디터 모드와 익스플로러(미리보기) 모드 모두에서 일관된 경험을 제공합니다.

---

## 2. 지원 카드 타입

| 타입 | 명칭 | 설명 | 마크다운 문법 |
| :--- | :--- | :--- | :--- |
| **Plain** | 기본 텍스트 | 일반적인 텍스트 링크 형태로 표시 | `[mx-plain#](url)` |
| **Link** | 단순 링크 | 사용자 정의 제목이 있는 밑줄 링크 | `[mx-link#제목](url)` |
| **Card** | 썸네일 카드 | 제목, 도메인, 썸네일 이미지가 포함된 가로형 카드 | `[mx-thumb#제목](url)` |
| **Video** | 비디오 플레이어 | 유튜브 URL 입력 시 사용 가능한 임베드 플레이어 | `[mx-video#제목](url)` |

---

## 3. 주요 기능 및 상호작용

### 3.1 에디터 모드 상호작용
- **타입 전환 메뉴**: 링크 카드를 클릭하면 나타나는 팝업 툴바에서 4가지 타입(Text, Link, Card, Video)을 자유롭게 전환할 수 있습니다.
- **실시간 편집**: '편집' 버튼을 통해 URL과 제목(Alt Text)을 즉시 수정할 수 있습니다.
- **클립보드 복사**: 툴바의 복사 아이콘을 클릭하여 URL을 간편하게 복사할 수 있습니다.
- **삭제**: 팝업 메뉴 하단의 '링크 제거' 버튼을 통해 노드를 삭제할 수 있습니다.

### 3.2 실시간 마크다운 변환 (Live Markdown)
- **노드 확장 (Expand)**: 커서가 링크 카드 노드에 위치하거나 닿으면, 편집을 위해 즉시 마크다운 텍스트(`[mx-...]`)로 변환됩니다.
- **노드 축소 (Collapse)**: 편집을 마치고 커서가 해당 영역을 벗어나면, 설정된 타입에 맞는 UI 카드로 즉시 자동 변환됩니다.
- **자동 인식**: 일반 URL(https://...)을 입력한 후 커서를 옮기면 자동으로 `Plain` 타입의 링크 카드로 변환됩니다.

### 3.3 익스플로러(미리보기) 모드
- 에디터에서 설정한 카드 타입이 미리보기 화면에서도 동일하게 렌더링됩니다.
- 유튜브 비디오의 경우 미리보기 화면에서 직접 재생이 가능합니다.
- 다크 모드/라이트 모드 테마에 맞춰 카드의 배경색과 텍스트 색상이 자동으로 조정됩니다.

---

## 4. 데이터 구조 및 직렬화
- **저장 형식**: 표준 마크다운 링크 문법에 `mx-` 접두사를 활용한 사용자 정의 스킴을 사용합니다.
- **예시**: `[mx-thumb#나의 여행 블로그](https://example.com/blog)`
- **처리 과정**: 
  1. `preprocessMarkdown`: 파일을 읽을 때 `<span>` 태그로 변환하여 에디터 노드로 로드.
  2. `postprocessMarkdown`: 저장 시 `[mx-...]` 텍스트 문법으로 복원.
