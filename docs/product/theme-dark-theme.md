# 프리미엄 다크 테마 명세 (Ultimate Obsidian Dark Mode)

## 1. 디자인 철학
'Mark Explorer'의 다크 테마는 단순한 색상 반전을 넘어, 장시간 문서 작업 시에도 눈의 피로를 최소화하면서도 현대적이고 고급스러운 감각을 전달하는 것을 목표로 합니다. **'Obsidian & Violet'** 컨셉을 바탕으로 시각적 깊이감과 정교한 마감 품질을 제공합니다.

## 2. 컬러 시스템 (Color Palette)
| 요소 | 색상 코드 | 설명 |
| :--- | :--- | :--- |
| **Primary Background** | `#0b0e14` | 깊이감 있는 흑연색 (Obsidian Black) |
| **Surface (Secondary)** | `#151921` | 레이어 구분을 위한 짙은 회색 (Slate) |
| **Accent (Primary)** | `#7c3aed` | 브랜드 정체성을 드러내는 선명한 보라색 (Vivid Violet) |
| **Text (Main)** | `#e2e8f0` | 눈이 편안한 부드러운 흰색 |
| **Text (Muted)** | `#94a3b8` | 보조 정보 및 메타데이터용 회색 |
| **Accent Glow** | `rgba(124, 58, 237, 0.1)` | 활성 항목 강조를 위한 미세한 보라색 광채 |

## 3. 주요 디자인 특징
### 3.1 글래스모피즘 (Glassmorphism)
- **적용 영역**: 웹(Web) 환경의 사이드바 및 탭 바.
- **효과**: `backdrop-filter: blur(12px)`와 반투명 배경색을 조합하여 레이어 간의 시각적 계층 구조를 명확히 하고 고급스러운 투명감을 부여합니다.

### 3.2 정교한 인디케이터
- **TOC 활성 항목**: 텍스트 왼쪽에 Violet 수직 바를 배치하고, 항목 배경에 부드러운 라운딩 처리를 더해 현재 위치를 직관적으로 표현합니다.
- **활성 탭**: 탭 상단에 Violet 라인을 추가하고 배경색을 Primary Background로 전환하여 집중도를 높입니다.

### 3.3 코드 블록 스타일링
- **배경**: 에디터 배경보다 더 깊은 블랙(`#0b0e14`)을 사용하여 코드 영역을 강조합니다.
- **하이라이팅**: `One Dark` 기반의 구문 강조를 적용하되, 배경색과의 대비를 조정하여 가독성을 극대화했습니다.

## 4. 기술 명세
- **테마 관리**: `contexts/ThemeContext.tsx` (전역 상태 관리)
- **상수 정의**: `constants/theme.ts` (다크/라이트 토큰 통합 관리)
- **컴포넌트 연동**: `useTheme` 커스텀 훅을 통한 동적 스타일링 적용.
