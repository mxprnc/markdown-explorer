# YouTube Playlist Exporter (Product Specification)

이 문서는 Mark Explorer의 **YouTube Playlist Exporter** 기능에 대한 상세 제품 명세를 다룹니다. 본 기능은 단순한 링크 추출을 넘어, 유튜브 플레이리스트를 체계적인 마크다운 문서로 변환하고 개인의 학습/참조 노트와 결합할 수 있도록 설계되었습니다.

---

## 1. 추출 모드 (Extraction Modes)

사용자의 용도에 따라 두 가지 핵심 추출 모드를 제공합니다.

### 1.1 D-1: Dedicated Note (전용 문서 모드)
- **개념**: 특정 플레이리스트만을 위한 독립적인 마크다운 문서를 생성합니다.
- **용도**: 온라인 강의 코스 정리, 특정 주제의 영상 아카이브 등.
- **마크다운 구조**:
  ```markdown
  ## 영상 제목
  [포맷에 따른 링크 내용]
  
  ### Note
  (사용자 메모 영역)
  
  ---
  ```
- **특징**: 각 영상이 하나의 섹션(`##`)이 되어 목차(TOC)에서 쉽게 탐색할 수 있습니다.
- **저장 규격**: `{플레이리스트 제목}_{HHmmss}.md` 파일로 자동 생성 및 저장.

### 1.2 D-2: Inline Mix (혼합 삽입 모드)
- **개념**: 현재 작성 중인 일반 문서 내에 플레이리스트 목록을 리스트 형태로 삽입합니다.
- **용도**: 블로그 포스트 내 참고 영상 목록, 짧은 레퍼런스 리스트 등.
- **마크다운 구조**:
  ```markdown
  - **영상 제목** ([포맷에 따른 링크])
    > (사용자 메모 영역)
  ```
- **특징**: 목차(TOC)를 오염시키지 않으면서 기존 문서의 흐름에 자연스럽게 통합됩니다.

## 2. 사용자 인터페이스 (Dialog Interface)

### 2.1 설정 (Settings Panel)
- **Link Format**: 추출될 링크의 시각적 형태를 결정합니다.
  - `Text`: 단순 제목 및 링크 텍스트.
  - `URL`: 클릭 가능한 원시 URL.
  - `Card`: 제목, 썸네일, 설명이 포함된 커스텀 카드(`[mx-thumb]`).
  - `Video`: 에디터 내에서 즉시 재생 가능한 임베드 플레이어(`[mx-video]`).
- **List Type**: 리스트의 마커 형식을 결정합니다.
  - `Numbered`: 숫자 리스트 (`1.`).
  - `Bulleted`: 점 리스트 (`-`).
  - `Plain` (**기본값**): 기호 없는 단순 개행 나열.
- **Item Limit**: 추출할 영상의 개수를 직접 입력합니다. (기본값: 20)

### 2.2 라이브 프리뷰 (Live Preview)
- **정직한 미리보기 (Honest Preview)**: 시스템은 추출 버튼을 누르기 전, 우측 패널에 실제 파일에 기록될 내용을 실시간으로 렌더링합니다.
- **탭 구성**:
  - `Markdown`: 실제 저장될 '원시 코드'를 확인하고 복사할 수 있습니다.
  - `Preview`: Mark Explorer의 렌더링 엔진을 통해 실제 '최종 결과물'의 시각적 모습을 확인합니다.

---

## 3. 추출 후 작업 및 편집 (Post-Export Workflow)

### 3.1 시각적 렌더링
- 추출된 마크다운은 에디터 및 프리뷰 모드에서 선택한 `Link Format`에 따라 즉시 시각화됩니다.
- 특히 `Card`와 `Video` 타입은 Mark Explorer 전용 노드로 변환되어 프리미엄한 미디이 경험을 제공합니다.

### 3.2 수동 편집 및 메모
- 사용자는 생성된 마크다운 내의 `### Note` 또는 인용구(`>`) 영역에 자유롭게 자신의 생각이나 학습 내용을 기록할 수 있습니다.
- **안전성**: 본 기능의 구조는 향후 '재추출(Re-extract)' 시에도 사용자가 작성한 메모 영역을 보존하며 영상 목록만 업데이트할 수 있도록 논리적으로 설계되었습니다.

---

## 4. 품질 및 기술 규격

- **YouTube API v3**: 실제 재생목록 데이터를 실시간으로 수집합니다.
- **자동화 검증**: Playwright를 통해 Chromium, Firefox, Webkit 환경에서 기능의 무결성을 100% 검증했습니다.
- **확장성**: 탐색기 폴더 우클릭 메뉴와 연동되어 진입 장벽을 최소화했습니다.

---

> [!TIP]
> **D-1 모드**는 대규모 학습 노트를 만들 때, **D-2 모드**는 가벼운 공유용 문서를 만들 때 최적의 성능을 발휘합니다.
