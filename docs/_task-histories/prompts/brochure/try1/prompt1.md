### 프롬프트

당신은 세계 최고의 제품 마케터이자 프론트엔드 개발자입니다. 'Mark Explorer'라는 프리미엄 마크다운 에디터 및 탐색기를 위한 홍보용 랜딩 페이지(단일 HTML 파일)를 제작해야 합니다.

#### 1. 제품 개요
- **제품명**: Mark Explorer (마크 익스플로러)
- **슬로건**: "AI-Powered Markdown Sidekick" - 당신의 로컬 문서를 지능적으로 탐색하고 관리하는 가장 완벽한 방법.
- **핵심 가치**: 로컬 마크다운 파일을 실시간으로 탐색하고, 현대적인 UI에서 스마트하게 관리하며, AI와 협업할 수 있는 강력한 도구입니다.

#### 2. 페이지 디자인 가이드라인 (중요)
- **프리미엄 미학**: Obsidian의 다크 모드 감성과 현대적인 보라색(Vivid Violet) 포인트를 사용하세요.
- **글래스모피즘**: 사이드바, 카드, 네비게이션 바에 `backdrop-filter: blur(12px)`와 반투명 배경을 적용하여 깊이감을 부여하세요.
- **색상 팔레트**:
  - Primary: `#7c3aed` (Violet)
  - Background: `#0b0e14` (Obsidian Black)
  - Surface: `#151921` (Slate)
  - Text: `#e2e8f0` (Soft White)
- **애니메이션**: 스크롤에 따른 페이드인 효과, 마우스 호버 시의 미세한 스케일 업 및 광채 효과를 추가하세요.
- **레이아웃**: 반응형 디자인(Mobile/PC)을 완벽하게 지원해야 합니다.

#### 3. 포함되어야 할 섹션 및 내용
1. **Hero Section**:
   - 매력적인 헤드라인과 서브 헤드라인.
   - "Get Started" 및 "View on GitHub" 버튼 (프리미엄 버튼 스타일).
   - 제품의 프리미엄한 느낌을 주는 목업 이미지 영역 (placeholder 대신 세련된 CSS 그래픽 추천).
2. **Core Features (Grid Layout)**:
   - **Live Editor**: Tiptap 엔진 기반의 강력한 실시간 편집 환경.
   - **Multi-pane Workspace**: 2x2 스플릿 뷰와 탭 드래그 앤 드롭으로 극대화된 생산성.
   - **Nextra Export**: 한 번의 클릭으로 로컬 문서를 세련된 정적 사이트로 변환.
   - **AI & MCP**: Gemini와 Model Context Protocol을 통한 지능적인 문서 작업 및 파일 시스템 제어.
   - **Offline-First**: 브라우저 API를 통한 로컬 파일 직접 관리와 IndexedDB 캐싱.
3. **Multi-Platform Support**:
   - Android, iOS, Windows, macOS, Linux를 모두 지원함을 아이콘과 함께 강조.
4. **Use Cases (Visual Cards)**:
   - 모바일에서의 편리한 마크다운 관리.
   - 유튜브 영상 요약 및 리서치 정리 도구.
   - Google Drive/iCloud 연동을 통한 클라우드 협업.
   - 등산, 여행 등 소중한 일상을 기록하는 프라이빗한 저널링.
   - 백업 걱정 없는 SNS/블로그 초안 작성 도구.
5. **Footer**:
   - 비영리 목적의 무료 배포(개인/기업 모두 자유롭게 사용 가능).
   - 저작권 정보 및 소셜 링크.

#### 4. 기술 요구사항
- **단일 파일**: HTML, CSS, JavaScript를 하나의 파일에 포함하세요.
- **폰트**: Google Fonts의 'Inter' 또는 'Outfit'을 사용하세요.
- **아이콘**: Phosphor Icons 또는 Lucide Icons CDN을 사용하세요.
- **가독성**: 텍스트 가독성을 최우선으로 하며, 타이포그래피 계층 구조를 명확히 하세요.

이제 사용자가 한 눈에 반할 수 있는, Apple이나 Stripe 수준의 고퀄리티 홍보 페이지 코드를 작성해 주세요.
