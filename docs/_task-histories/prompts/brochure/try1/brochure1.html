혁신적인 기능들

단순한 에디터를 넘어, 당신의 워크플로우를 완성합니다.

Live Editor

Tiptap 엔진 기반의 강력한 실시간 편집 환경. 작성과 동시에 완성되는 아름다운 마크다운을 경험하세요.

Multi-pane Workspace

2x2 스플릿 뷰와 자유로운 탭 드래그 앤 드롭. 여러 문서를 동시에 비교하고 편집하며 생산성을 극대화하세요.

Nextra Export

한 번의 클릭으로 로컬 문서를 세련된 정적 사이트로 변환. 별도의 설정 없이 전문가 수준의 문서 사이트를 구축하세요.

AI & MCP Integration

Gemini와 Model Context Protocol 연동. AI가 당신의 문서 맥락을 이해하고 파일 시스템을 직접 제어합니다.

Offline-First

브라우저 Native API를 통한 로컬 파일 직접 관리. 데이터는 항상 당신의 장치에 안전하게 머뭅니다.

Premium Design

Obsidian 스타일의 다크 모드와 글래스모피즘 UI. 장시간 작업에도 눈이 편안하고 아름다운 작업 환경을 제공합니다.

다양한 사용 사례

Mark Explorer는 모든 상황에서 최고의 동반자가 됩니다.

안드로이드/iOS 폰에서 마크다운 파일을 쉽고 빠르게 편집하고 관리하세요.

🔍 리서치 및 요약

유튜브 영상이나 뉴스 내용을 AI와 함께 요약하고 탐색기로 체계적으로 정리하세요.

☁️ 클라우드 동기화

등산 후의 맑은 정신, 여행의 감동을 마크다운 문서로 프라이빗하게 남기세요.

✍️ 소셜 초안 작성
